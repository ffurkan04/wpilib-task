# Cymurghs 7466 Programming WPI LIB Task
## Robotun içeriği
### Drivebae:
  + Tank drive sürüş yapısına sahip
  + FRC'nin kitinden çıkan motorları kullanır
  + Şaşi, tekerlek, güç aktarım mekanizmaları vardır
### Elektrik ve güç sistemi:
  + RoboRIO(Robotun beyni)
  + PDH (Power Distribution Hub robotta elektrik gücünü dağıtır)
  + Motor Kontrolcüleri
  + Batarya ve sigorta
### Oyun parçaları: 
  Sezona göre değişen oyun parçalarını kontrol etmek için kullanırız 
  + Intake mekanizması
  + Motor ve dişli/kayış aktarımı
### Sensötler ve Sürücü kontrolleri: 
  + Joystick veya gamepad 
  + Encoder
### Yazılım altyapısı: 
  + WPI Lib Java kullanılmaktadır
  + Command-Based yapılıdır
  + Command'lardan ve Subsystemlardan oluşur.
